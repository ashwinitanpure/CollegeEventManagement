Overview :


Android Application Development workshop is conceptualized by some top industry professionals in association with HT India Labs. It is going to be India's first & biggest workshop series based on this market flag bearer of cloud systems. This workshop will also provide a platform where young engineers can mould their imagination into reality and feel the excitement first hand.

Course :

Day 1:

  • Introduction:
  • What is Aerodynamics?
  • Skills required to build Aerodynamics model.
  • Fundamental of Aerodynamics.
  • Microcontroller and IO hardware
  • Introduction to Nano-Drone.
  • Structure Building of Drone.
  • Types of Communication Protocol
  • Introduction of Wi-Fi Communication Technology

Day 2:

  • Introduction to Drone Sensor
  • Coding for Sensor data.
  • Coding for full Drone.
  • Different flying modes in droneCode for Pit, Roll, yaw Stable Position
  • Coding for Take-off and landing of Drone
  • On Board LED Control Coding.
  • Drone Competition Q & A Session on Aerodynamics System

Benefits :

  • Student will be clear in Fundamental of Aerodynamics
  • Drone Assembly and Programming Student will understand physics behind the drone design
  • Student will be able to interface various devices with Microcontroller
  • Student will learn how to control things via Wi-Fi
  • Student will be able to develop their own Aerodynamics System
